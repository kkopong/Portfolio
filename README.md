# Evolving Engineer Portfolio

A modern, story-driven personal portfolio designed to showcase the journey of an upcoming IT professional. This project moves away from traditional static resumes to create a "building in public" experience that emphasizes growth, curiosity, and technical evolution.

![Portfolio Preview](/Users/kukua/.gemini/antigravity/brain/370e460e-a4e0-40a8-9a5b-7495f8e47df7/full_portfolio_check_1773749138440.png)

## 🚀 Vision: "The Evolving Engineer"
This portfolio is built around the philosophy of transparency and continuous learning. It doesn't just show what has been mastered, but also what is currently being "conquered."

## 🛠️ Tech Stack
- **Framework**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## ✨ Key Features
- **Narrative Hero Section**: A personalized introduction focusing on the "why" and location (Accra, Ghana).
- **Growth Evolution Timeline**: A vertical, chronological journey of technical milestones from 2022 to the present.
- **Tech Arsenal**: Categorized skills (Foundations, Cloud, AI) with status indicators for ongoing learning.
- **The Personal Lab**: A dedicated space for experimental projects, research modules, and work-in-progress.
- **Selected Operations**: A clean grid of production-ready applications with direct GitHub integration.
- **CV Integration**: Automated download link for the latest Resume/CV.
- **Atmospheric UI**: A custom "Cyber-Grid" design system with dynamic mouse-following glow effects and smooth glassmorphism.

## 🏁 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kkopong/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```

### Build
Create a production-ready build:
```bash
npm run build
```
